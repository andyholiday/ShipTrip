+ Bottom-Sheet

Durchgehende Designsprache

Ein System — 5 Tabs

Tokens, Strukturprinzipien, Radius-Hierarchie

Farb-Tokens (unveraendert)
oceanBlue #0C8CE9
Primär, Chips aktiv, Bar-Charts
oceanLight #36A9F0
Seetag-Pins, sekundäre Routen
sunsetOrange #FF6B35
Start-Pin, Rabatt-Badge, Upcoming
seaGreen #34C759
End-Pin, Port-Preview, Rating
navyDark #1A365D
Profil-Header-Gradient Start
Strukturprinzipien
Geo-Kontext überall
Mini-Route-SVG in Zeile, GeoFallback in Hero, Weltkarte in Stats — kein Tab ohne geografischen Anker
Radius-Hierarchie beibehalten
r16 Hero · r12 cardStyle · r10 Row/Strip · r6 Thumbnail — unverändert aus bestehendem System
Dark-first, kein weisser Hintergrund
Karte lebt im Dunkel. Alle Cards: rgba(255,255,255,0.05-0.08) statt systemGroupedBackground
Bottom-Sheet statt Tab-Wechsel
Hafen-Detail, Deal-Detail: Sheet über Karte/Liste statt NavigationLink — Kontext bleibt sichtbar
Interaktions-Vokabular
Tap Chip → Fokus-Zoom Karte
Tap Pin → Bottom-Sheet (Port-Detail)
Long-press Row → Kontextmenu (Löschen/Teilen)
Swipe-left Deal-Row → Löschen (bestehend)
